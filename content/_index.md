## How the mock content works

Each module page is a Hugo section page. The cards below each module come from normal Markdown article bundles with front matter for `tags`, `categories`, `series`, and `series_order`.

Inside articles, `##` and `###` headings automatically build the table of contents when `showTableOfContents` is enabled.
